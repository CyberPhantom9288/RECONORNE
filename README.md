# 🚀 RECONORNE — Ultimate OSINT & Reconnaissance Tool
---

## 🔎 What is RECONORNE?

RECONORNE is a powerful and efficient OSINT (Open Source Intelligence) & reconnaissance tool tailored for cybersecurity professionals and ethical hackers. It automates the collection of critical information from various public sources to aid in penetration testing, vulnerability analysis, and digital investigations.

---

## ⚡ Features

- ✅ Automated domain, IP, email, and username reconnaissance  
- ✅ Fast and reliable data gathering from multiple public sources  
- ✅ Simple and intuitive command-line interface (CLI)  
- ✅ Easily extensible to add more data sources  
- ✅ Written in Python for easy customization and integration  

---

## 🛠️ Installation

```bash
git clone https://github.com/CyberPhantom9288/RECONORNE.git
cd RECONORNE
pip install -r requirements.txt
python3 RECONORNE.py
```
